## iniciar proyecto

- npm init -y

## dependencias

- npm install nodemon
- npm i express dotenv
- npm i mongoose
- npm i bcryptjs
- npm i express-validator

## correr proyecto

- npm i
- nodemon index

## Comandos git

- git add .
- git commit -m "first commit"
- git push -u origin main
- git tag -a v1.0.0 -m "Creando tag"
- git push --tags
